# Passive Market Instability Extension

This is a notebook-first research package for extending "A Model for Passive That Breaks the Market" by Michael Green, Hari P. Krishnan, and Stephan Sturm. It is not a production QuantConnect strategy and it does not place trades.

## Model

The paper models market value `S(t)` as mean reverting toward a smooth fundamental value `F(t)`, with mean reversion weakening as passive share `p(t)` rises:

```text
dS(t) =
    kappa * (1 - p(t)) * (F(t) - S(t)) dt
    + sigma * sqrt(F(t) * S(t)) dW(t)
```

The notebook uses the paper-style assumptions `F(t) = F0 * exp(r * t)` and `p(t) = 1 / (1 + exp(-alpha * (t - t0)))`, then compares baseline and sensitivity cases.

## Data-Source Ideas

The baseline passive-share curve uses the paper's Haddad, Huebner, and Loualiche 2025 interpretation, with approximate logistic growth parameter `alpha = 0.106`.
Brightman and Harvey 2025 is used as a close alternative passive-share curve, with approximate logistic growth parameter `alpha = 0.100`.
The David Dredge / Yahoo Finance active-passive chart is used only as a visual benchmark unless someone manually digitizes the chart and labels it as manual data.

This notebook uses three levels of data quality. Official data is downloaded directly from an authoritative source. Manual data is hand-entered or digitized from a paper, chart, appendix, or exhibit. Proxy data approximates a target variable when the true data is unavailable. ETF volume times return is a flow-pressure proxy, not an official ETF flow series. Estimated index weights from market capitalization are not the same as official S&P 500 index weights.

## Notebook Sections

1. Introduction to the paper's model and the three extensions.
2. Paper parameter display and instability threshold checks.
3. Passive-share curve comparison across Haddad baseline, Brightman-Harvey, conservative, aggressive, and no-passive cases.
4. Paper-style simulation sanity checks for no passive share and rising passive share.
5. ETF flow-pressure proxy analysis for `SPY`, `IVV`, `VOO`, and `VTI`.
6. Cross-sectional passive-pressure diagnostics for a small demo stock universe.
7. Final sanity-check summary table.

## Data Limitations

The first version is intentionally transparent about data quality. It does not claim to have official ETF creations/redemptions, official S&P 500 constituent weights, or machine-readable David Dredge / Yahoo chart data. The repository includes committed processed snapshots under `infrastructure/pipelines/{passive_share,etf_flows,equity_liquidity}/data/processed/` so the notebook can run without immediately fetching the network. If those snapshots are removed and `yfinance` is unavailable or internet access fails, the notebook keeps running with clearly labeled placeholders or synthetic demo panels so the research workflow remains inspectable.

The cross-sectional universe is only a demonstration set:

```text
AAPL, MSFT, NVDA, AMZN, META, GOOGL, BRK-B, LLY, JPM, XOM
```

Do not interpret the demo universe as a full S&P 500 result.
